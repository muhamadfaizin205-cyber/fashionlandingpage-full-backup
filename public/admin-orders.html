# PRODUCT REQUIREMENTS DOCUMENT (PRD): PromoGen AI *Version:* 1.0.0 *Status:* Initial Scaffolding Phase *Stack:* Next.js 14+, TypeScript, Supabase, Tailwind CSS ## 1. Executive Summary PromoGen AI adalah aplikasi internal-tool berbasis web untuk mengotomatisasi pembuatan salinan promosi hotel. Aplikasi ini mengintegrasikan database terpusat dengan logika bisnis deterministik untuk menghasilkan materi iklan yang konsisten, akurat, dan sesuai dengan standar perusahaan. ## 2. System Architecture * *Frontend:* Next.js (App Router) & Tailwind CSS. * *Backend:* Next.js API Routes (Serverless Functions). * *Database:* Supabase (PostgreSQL) sebagai Single Source of Truth. * *Deployment:* Vercel (Auto-deploy via GitHub). ## 3. Data Schema (Supabase PostgreSQL) Tabel: hotels | Column | Type | Constraints | Description | |---|---|---|---| | id | UUID | Primary Key | Identitas unik properti. | | name | Text | Unique, Not Null | Nama properti (e.g., RedDoorz near X). | | price_reddoorz | Integer | Nullable | Harga dasar RedDoorz Room. | | price_superior | Integer | Nullable | Harga dasar Superior Room. | | price_double | Integer | Nullable | Harga dasar Double Room. | | address_raw | Text | Nullable | Alamat lengkap properti. | | is_active | Boolean | Default: true | Status ketersediaan properti. | ## 4. Business Logic & Pricing Engine Perhitungan dilakukan di server-side (/api/generate) untuk menjaga keamanan. ### 4.1. Pricing Algorithm 1. base_price = price_reddoorz (if not null), ELSE price_superior (if not null), ELSE price_double. 2. *Final Calculation:* final_price = base_price - 10000. ### 4.2. Title Transformation * *IF* name contains "near": dynamic_title = "Penginapan Murah Dekat [Extracted Value]" * *ELSE*: dynamic_title = "Penginapan Murah di [City/Region]" ### 4.3. Template Structure Sistem menghasilkan output dengan format eksak: text {{dynamic_title}} {{name}} Hotel murah, aman, dan nyaman Harga mulai dari {{final_price}}/kamar/malam Minimal 18 tahun ber-KTP ✅ Bisa bayar cash di hotel ✅ Bisa untuk pasutri dan keluarga ✅ Bisa untuk rombongan ✅ Fasilitas: resepsionis, alat mandi gratis, toilet di dalam kamar, AC, TV, dan wifi 📍{{address_raw}} Booking sekarang screenshot dan kirim ke WA 0831-3153-3097! Atau booking via aplikasi RedDoorz ada HARGA SPESIAL DIPOTONG 25% pake kode promo RDAFFASIKPROMO. Kode promo bisa dipake berulang kali! ## 5. UI/UX Design Standards (Design System) * *Typography:* Geist Sans / Inter. * *Color Palette:* Zinc/Slate (Monochromatic) dengan Indigo (#6366f1) sebagai Primary Accent. * *Iconography:* lucide-react (SVG). Dilarang menggunakan emoji. * *Component Library:* shadcn/ui. * *Layout:* Desktop Split-view (Kiri: Form Input, Kanan: Live Preview). ## 6. Security & Compliance 1. *Credential Management:* Gunakan .env dan Environment Variables Vercel. Jangan hardcode API Key. 2. *Service Role Key:* Gunakan SUPABASE_SERVICE_ROLE_KEY hanya di Serverless Functions. Dilarang keras mengekspos di sisi klien. 3. *Validation:* Sanitasi input string dari user sebelum query database untuk mencegah SQL Injection. ## 7. Roadmap * *Phase 1:* Setup Repository & Database Schema. * *Phase 2:* Implementasi API Route (Pricing Engine & Template Logic). * *Phase 3:* Development UI (Shadcn/ui & Responsiveness). * *Phase 4:* Data Enrichment (Google Places API). * *Phase 5:* Deployment & Verification. Dokumen teknis acuan pengembangan.
